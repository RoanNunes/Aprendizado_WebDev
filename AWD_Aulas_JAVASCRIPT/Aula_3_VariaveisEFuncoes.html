Variaveis e Funções

A base de uso uma linguagem de programação se consiste em saber como e onde utilizar variáveis e funções.
Toda variável deve ser declarada pelo desenvolvedor em seu código antes de seu uso.
Há funções presente em Javascript, no entanto elas são descritas por keywords ou palavras chaves.
Toda função criada pelo desenvolvedor deve ser declarada, determinar a necessidade de argumentos ou quem são eles e bloco de instruções que vão ser executados.
É fundamental ver o código fonte das aulas a seguir para ver os scripts em ação. Quando possível ele será descrito.

Variáveis (ou variables)

Variáveis são espaços na memória declarados a armazenar algum dado.
Em Javascript podemos declarar variáveis de três métodos:

  • var - Usada desde 1995, declara toda e qualquer variável;
  • let - Usada desde 2015, declara toda e qualquer variável;
  • const - Usada desde 2015, declara toda e qualquer variável a qual não é possível alterar, por isso "constante";

Qualquer método pode usado para declarar uma variável.
Entre 'var' e 'let', recomenda-se 'let' mas nada impede de usar 'var';
Entre 'var' e 'const', recomenda-se 'const' mas nada impede de usar 'var'. No entanto, quando o contexto é para declarar uma constante imutável, utilize a declaração 'const'.
A declaração 'var' não possui limitação, designação ou tipo que precisa ser definido para sua utilização.
Variáveis podem ser definidas em sua declaração ou indefinidadas (exceto 'const').
Em uma declaração é possível declarar (e definir) multiplas variáveis, sendo separadas por (,) virgula.
Uma variável pode assumir qualquer tipo de dado permitido pelo Javascript, não havendo a necessidade declarar o tipo dela.
Segue alguns exemplos:

  • 'var nome = "Ronaldo";'
  • 'var nomeRequesitado;'
  • 'let preco = 12,99;'
  • 'let valorBase = 9,99, valorGoiaba = 21,99, valorBanana = 14,99 , valorAbacaxi = 100;'

Variáveis tem ou o escopo global, ou escopo local.

  • Variável global - Se permanecem vivas até o momento que a página é descartada (exemplo: quando fecha o navegador ou muda de página);
  • Variável local - Se permanecem vivas enquanto a função estive em execução. Quando finalizada a variável deixa de existir;
    • Por boa prática, recomenda-se:

    • De o nome a variáveis. Tais nomes devem ser lógicos ao propósito dela.
    • Não use keywords como nome de variáveis.
    • Não comece o nome de variáveis com números.
    • Não use espaços ao nomear variáveis.
    • Recomendação os modelos de nomes: 'minhaVariavelPreferedia' ou 'minha_variavel_preferida'

    Funções ou (functions)

    Funções, são blocos de instruções declarados pela keyword 'function' os quais podem ser usados em meio o script ou em meio o arquivo html.
    Funções tem objetivo de permitir a reutilização de código (ou instruções) em meio ao programa.
    A sintaxe de uma função é a seguinte:
    function name(parameter1, parameter2, parameter3) { // code to be executed }
    Dentro de uma função, é possível criar variáveis locais as quais só podem ser usadas enquanto a função se executa.
