Sintaxe e Estrutura

Javascript é uma linguagem de programação interpretada, ou seja, possui toda complexidade de uma linguagem de programação mas a execução difere de uma linguagem que é compilada.
Javascript possui sua própria nomeclatura (ou linguagem) de realizar comandos e estruturar funções. Ele pode ser executado independente da presença de HTML ou/e CSS.
Com passar dos anos, Javascript tornou realidade a existencia de "Webapps", aplicações completas que rodam no próprio navegador, ou solicitam comandos/funções ao servidor realizar para certa aplicação.

Sintaxe

Temos uma série de elementos em Javascript.
Javascript é uma linguagem baseada no Inglês americano, então segue os termos usados:

  • Values (ou Valores)
  • Variáveis (ou Variables)
  • Operador (ou Operators)
  • Keyword (ou Palavra-chave)
  • Function (ou Funções)
  • Comments (ou Comentário)

Segue algumas regras de formatação:

  • Números são escritos sem ou com decimais.
    • [EXEMPLO] 10.1 ; 25 ; 47 ; 56,32 ; 146416947,13;
  • Textos (ou Strings) são escritas dentro de aspas duplas ("") ou aspas simples ('');
    • [EXEMPLO] "Olá" ; 'Fernando estava aqui!' ; "Quem sou eu" ; 'Número 37 é o ganhador'.
  • JavaScript é 'case sensitive', isto é, ele faz diferenciação do caractere minúsculo e maiúsculo.
    • [EXEMPLO] 'nomeProfissional' é diferente de 'nomeprofissional';
  • Javascript não aceita o hifen (-), diferente de HTML e CSS.
  • Após cada linha de comando, ela é finalizada por ponto e virgula (;).

Estrutura

Cada elemento presente no Javascript possui uma estrutura para seu uso.

  • Values (ou Valores)
  • Variables (ou Variáveis)
  • Operators (ou Operadores)
  • Keyword (ou Palavra-chave)
  • Function (ou Funções)
  • Comments (ou Comentário)

Values (ou Valores)

São os valores númericos definidos.
Strings também podem ser pré-definidas, mas usualmente são variáveis.
Segue os exemplos:

  • 10
  • 47
  • 56,32
  • 146416947,13
  • "Eu sou uma string definida"
    • Isso é o exemplo de uma string definida.

Variables (ou Variáveis)

Variáveis são containers que armazenam dados na memória.
Em Javascript, para a declaração de uma variável podemos utilizar: 'var', 'let' ou 'cont';
Entre eles, tais possuem objetivos diferentes, será explicado em um futuro momento;
Segue exemplos de seu uso:

  • var x = 10;
  • let y; y = 47;
  • const raio = 56,32;
  • var u, carro, horarioPartida;
    • É possível declarar várias variaveis em uma única linha.

Operators (ou Operadores)

Os Operadores matemáticos usualmente encontrados numa calculadora, estão presentes em operadores de programação. No entanto, há mais operadores do que somente os matemáticos. Sua distinção normalmente é a combinação de vários simbolos matemáticos para sinalizar outra operação.
Os tipos de operadores de programação seriam:

  • Operadores aritméticos
    • São operadores matemáticos que sinalizam alguma operação de cálculo.
  • Operadores de atribuição
    • Semelhantes aos operadores matemáticos, no entanto acrescentam a função de sinalizar a atribuição de um valor ao outro.
    [EXEMPLO] x = y ; x += y (que seria igual a (x = x + y)) ; x /= y (que seria igual a (x = x / y)) ;
  • Operadores de Comparação
    • Utilizados geralmente por funções específicas, com objetivo de comparar dois valores dado um contexto.
    [EXEMPLO] x == y (x é igual a y ?) ; x != y (x é diferente de y ?) ; x > y (x é maior que y ?) ;
  • Operadores Lógicos
    • Utilizados geralmente por funções específicas, sinalizar um contexto ou contextos
    [EXEMPLO] while(x!=y || x > 0) descreveria (Enquanto x for diferente de y OU x for maior que 0)
  • Operadores de Tipo
    • Sinalizam ou verificam o tipo ou instanciamento de uma variável ou função

Veremos mais sobre estes os operadores ao longo das aulas sendo usados dado a necessidade.
No entanto, aqui especificamente apenas será mostrado os aritméticos presentes em Javascript.
Eles seriam:

  • (+) - Soma ou somatório
  • (-) - Subtração
  • (*) - Multiplicação
  • (/) - Divisão
  • (**) - Exponêncial
  • (%) - Retorna o resto de uma divisão

Através de tais operadores é possível criar expressões. Quase todo método de expressar uma equação matemática pode ser descrita em código como expressões.
Algumas delas seriam:

  • 1 + 1;
  • 27 - 11;
  • 11 + 3 * 2;
  • (24 - 6) / 3;
  • 10 / 2;

Keyword (ou Palavra-chave)

São comandos prédefinidos pela linguagem o qual possuem uma nomeclatura própria e em alguns casos requerem a atribuição de valores. Ao longo das aulas será visto vários keyword sendo usado para realizar algumas função.
Algumas delas seriam:

  • let - Declara um bloco de variáveis.
  • var - Declara uma variável
  • for - Declara a marcação de um loop, determinados por um conjunto de declarações
  • switch - Declara um bloco de casos dado um contexto, e um tratamento para cada caso especificado
  • function - Declara uma função
  • ...

Existem vários keywords com funções específicas. Não seram mostrados todas elas, algumas serão destacadas ao longo das aulas.

Function (ou Funções)

São blocos de instruções (ou comandos) que podem ser utilizados ao longo do código, sejam eles prédefinidos
Iremos falar mais sobre em outra aula, no entanto segue a estrutura básica de uma função:

function nomeFuncao (argumento1, argumento2){ return resposta; }

Comments (ou Comentário)

São blocos de texto os quais não são interpretados pela linguagem, possuem objetivo de realizar anotações ou explicar em texto corrido o que está a ocorrer criadas pelo Desenvolvedor.
É importante a criação de comentários em meio ao código, para organiza ele e caso nessecidade explicar o que está a ocorrer.
Todo comentário é iniciado por (//) duas contrabarras ou pela contenção através de (/* */):

//Isso é um comentário em Javascript.
//Diferente de HTML e CSS!
/*
Isso é um comentário em Javascript!
Mais semelhante ao usado em CSS.
*/
