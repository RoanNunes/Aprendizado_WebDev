Javascipt!

Popularmente conhecido como "Javascript", originalmente conhecido como "Livescript" e oficialmente conhecido como "ECMAScript".
Javascript é a linguagem de programação a qual empodera as páginas e aplicações Web. Ela é uma linguagem de programação interpretada, diferente de C, C++, Java (e etc) que são linguagens compiladas.
Atualmente (2023), frameworks Javascript deixaram de ser somente voltados a front-end. É possível ver seu uso e frameworks dedicados a tarefas especializadas tanto no front-end quanto no back-end.
Aqui nós vamos passar a explorar o Javascript (ou ECMAScript) puro, de sua base à conhecimentos mais complexos.
Aqui a intenção será de apresentar uma base sólida para torna-ló independente ao pesquisar por mais conhecimento na área.
[AVISO] Por ser uma linguagem de programação, se torna impossível demonstrar e explicar cada interação e caso de uso que a linguagem permite executar.
[AVISO] Por ser uma linguagem de programação, recomendo a visualização do código fonte dos arquivos e a utilização das ferramentas de desenvolvedor no seu navegador (F12).
[AVISO] Para uma visualização mais didática, a maioria dos scripts criados seram executados a partir de botões. Caso haja um sendo executado automaticamente, será avisado.

Métodos de declara-lo

Temos os seguintes métodos:

  • Usando a tag HTML (script) para declarar o campo de um script que irá ser executado em meio HTML.
    • [EXEMPLO] "(script)Date();(/script)"
  • Usando a tag HTML (script) com a propriedade 'src', declarando o arquivo (e o local dele) o qual possui o script e usar suas funções.
    • [EXEMPLO] "(script src="meuscript.js")MinhaFuncao();(/script)"
