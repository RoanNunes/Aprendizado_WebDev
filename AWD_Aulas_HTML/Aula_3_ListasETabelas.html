Sobre Listas e Tabelas!

Oficialmente existem tags para criação de listas não ordenadas, listas ordenadas, listas descritivas e tabelas.
É possivel a criação de listas dentro de listas, o navegador já realiza esse tratamento, isto é, esse encadeamento de listas.

Tópico: Listas

Há uma série de tags responsabilizadas para criação de listas e cada uma tem um objetivo. Entenda que há a tag de lista; tag de item de lista; tag de descrição de lista. 

        tag (ul) Define uma lista não ordenada
        tag (ol) Define uma lista ordenada
        tag (li) Define um item de lista
        tag (dl) Define uma lista de descrição
        tag (dt) Define um termo em uma lista de descrição
        tag (dd) Descreve o termo em uma lista de descrição

Tópico: Tabelas

A criação de tabelas consiste nas seguintes etapas respectivamente: declaração da tabela, declaração de linha de tabela, declaração de célula da tabela.

        tag (tabela) define uma tabela.
        tag (th) define uma célula de cabeçalho em uma mesa.
        tag (tr) define uma linha em uma tabela.
        tag (Td) define uma célula em uma tabela.
        tag (geption) define uma legenda de tabela.
        tag (Colgroup) Especifica um grupo de uma ou mais colunas em uma tabela para formatação.
        tag (Col) Especifica as propriedades da coluna para cada coluna dentro de um elemento (Colgroup).
        tag (Thead) agrupa o conteúdo do cabeçalho em uma tabela.
        tag (Tbody) agrupa o conteúdo do corpo em uma tabela.
        tag (tfoot) agrupa o conteúdo do rodapé em uma tabela.

Em sua formatação mais básica, consiste em: table (table), tablerow (tr), table cell (ou data) (td)

Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Centro comercial Moctezuma Francisco Chang Mexico

Em sua formatação mais intermediarias, consiste em: table (table), table row (tr), table header (th), table row (tr), table cell (ou data) (td)

Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Centro comercial Moctezuma Francisco Chang Mexico

Em sua formatação mais complexas, consiste na divisão em seções a tabela, determinação de colunas e outros

Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Freds Futter Mia Aners China
Alf Kiste Mar Ders USA
