Sobre Hyperlinks e Imagens

Embora pareção diferentes, hyperlinks é tópico que abrange múltiplas interações com outras tags, além de ter sua própria.
Por isso, ela será estudada junto a tag (img) usada para referência arquivos de imagens interna ou externamente.

Tópico: Hyperlink

Os links HTML são hiperlinks, descritos pela tag (a). Você pode clicar em um link e pular para outro documento.
Casualmente conhecido quando você move o mouse sobre um link, a seta do mouse se transforma em uma pequena mão.
Lembrando que um link pode ser qualquer coisa, não limitando-se a somente texto.

Um exemplo de somente o link seria:

Um link vazio

Existe a distinção entre links internos e externos.

  • Um link interno é referênciado pelo posicionado dos arquivos a partir do arquivo .html
  • Um link externo é referênciado por uma URL ou endereço IP (ou até mesmo um arquivo em servidor)

Um exemplo link seria:

Um link para o Google!
Isso é um link diferenciado.

Além da distinção de links, há diferentes métodos de referência um link:

  • Absoluto - Um link global, o qual é externo a página (ou arquivo) a página a executar.
  • Relative - Um link local, o qual referência a uma página (ou arquivo) dentro do local referente da página a executar.

Tópico: Imagens

Imagens são referênciados pela tag (img) e possuem alguns atributos específicos dele.
A formatação de imagens em html não existe como em texto. Toda eslização é feita por CSS ou "style".
A tag tem dois atributos obrigatórios:

  • src - Especifica o caminho para a imagem
  • alt - Especifica um texto alternativo para a imagem

    • Existe outro métodos além do uso da tag (img), eu vou destacar o uso de (picture):
    Diferente de (img), a tag (picture) é utilizada para atribuir diferentes imagens e exibilas de acordo onde a mídia é exibida.
    O elemento (picture) contém um ou mais elementos (source), cada um referindo-se a diferentes imagens por meio do atributo srcset. Desta forma, o navegador pode escolher a imagem que melhor se adapta à visualização e/ou dispositivo atual.
    Cada elemento (source) possui um atributo media que define quando a imagem é a mais adequada.
