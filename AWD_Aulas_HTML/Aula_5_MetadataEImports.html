Sobre Metadata e Imports

"Metadados, ou Metainformação, são dados sobre outros dados. Um item de um metadado pode dizer do que se trata aquele dado, geralmente uma informação inteligível por um computador. Os metadados facilitam o entendimento dos relacionamentos e a utilidade das informações dos dados."
- Wikipedia 2023

Tópico: Metadados

Os metadados em maioria ficam posicionados no (head) do arquivo .html.
Por isso, veja o código fonte documento para melhor entendimento.
Segue a lista de tags mais usadas:

  • tag (head) Define informações sobre o documento
  • tag (title) Define o título de um documento
  • tag (base) Define um endereço padrão ou um destino padrão para todos os links em uma página
  • tag (link) Define a relação entre um documento e um recurso externo
  • tag (meta) Define metadados sobre um documento HTML
  • tag (script) Define um script do lado do cliente
  • tag (style) Define informações de estilo para um documento
