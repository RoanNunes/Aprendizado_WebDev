Formatação de TEXTO!

O que fazer com textos que são formatados de jeitos específicos? Eu já disse isso antes, mas agora há importância para os metadados. Podem ser usadas em meio a qualquer tag, esteja ciente disso. 

         - Bold text 
         - Important text 
         - Italic text 
         - Emphasized text 
         - Marked text 
         - Smaller text 
         - Deleted text 
         - Inserted text 
         - Subscript text 
         - Superscript text

- Bold text - Important text - Italic text - Emphasized text - Marked text - Smaller text - Deleted text - Inserted text - Subscript text - Superscript text

Text - Bold text -
O elemento HTML (b) define o texto em negrito, sem nenhuma importância extra.
Text - Important text -
O elemento HTML (strong) define o texto com forte importância. O conteúdo interno geralmente é exibido em negrito.
Text - Italic text -
O elemento HTML (i) define uma parte do texto em uma voz ou humor alternativo. O conteúdo interno geralmente é exibido em itálico. A tag (i) é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc.
Text - Emphasized text -
O elemento HTML (em) define o texto enfatizado. O conteúdo interno é normalmente exibido em itálico. Um leitor de tela pronunciará as palavras em (em) com ênfase, usando tonicidade verbal.
Text - Marked text -
O elemento HTML (mark) define o texto que deve ser marcado ou destacado.
Text - Smaller text -
O elemento HTML (small) define um texto menor.
Text - Deleted text -
O elemento HTML (del) define o texto que foi excluído de um documento. Os navegadores geralmente traçam uma linha através do texto excluído.
Text - Inserted text -
O elemento HTML (ins) define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido:
Text - Subscript text -
O elemento HTML (sub) define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O.
Text - Superscript text -
O elemento HTML define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW[1]:
