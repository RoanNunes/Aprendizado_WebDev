Olha, CSS é um assunto vasto.

Há múltiplos conceitos teoricos, tecnicas oficiais, métodos não tão oficiais, muito que poderia ser ensinado.
No entanto, aqui daremos um foco na manipulação, nomeclatura, destacando de vez em quando alguma propriedade específica.
Vou acresecentar que para lições de CSS é fundamental ler o código fonte, pois ele provavelmente terá seu arquivo dedicado.
Também poderá perceber que conforme as aulas progridem, o visual da página irá mudar.

Sintaxe (ou Syntax)

h4 { color: white; background-color: black; }

Temos o seletor: (h4);
Temos as propriedads: (color), (background-color);
Temos os valores: (white), (black);

Veremos que seletores podem ser: tag(s) html, class(s), id(s), pseudo-class(s), pseudo-element(s).
Cada um possui sua simbologia e nomeclatura, e por eles existem um conjunto de propriedades que podemos modificar.

Métodos de CSS

Há três métodos oficiais de inserir e utilizar CSS em um arquivo HTML:

  • Inline CSS - Utilizando o atributo 'style' nas tags HTMLs;
  • Interno CSS - Utilizando a tag 'style' dentro do arquivo HTML, geralmente entre as tags (head) e (body);
  • Externo CSS - Utilizando um arquivo distinto (.css), importanto ele a página HTML o qual usará ele;
